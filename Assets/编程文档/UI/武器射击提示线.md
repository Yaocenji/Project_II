
# 脚本 ProjectII.Weapon.ShootInstructionLine
继承：MonoBehaviour

UI中的武器提示线管理器。
这个脚本放在场景中的一个任意gameobject上边，但是并不放在任何Canvas下的UI物体上。
将两个image作为两条线，一次性设置起点和终点，然后自动调整其的正确位置

## 公有变量

可拖入的两个Image UI元素；

## 公有变量 宽度

## 方法 SetPosition0
输入两个参数：世界空间位置，表示起点和终点
设置image0的transform

## 方法 SetPosition1
输入两个参数：世界空间位置，表示起点和终点
设置image1的transform
