# 游戏世界观与前期剧本设定

## 一、 核心概念与基调
- **题材风格**：现代中式都市怪谈 + 改良克苏鲁（新怪谭）。
- **核心体验**：日常事物的异化、信息茧房下的规则怪谈、极度压抑的生存探索。
- **视角与表现**：纯 90° 俯视角。利用视野盲区（PlayerVision）、光影切割、FMOD 动态音效以及 UI 遮罩（手机、猫眼）来营造恐怖感，避免过早出现具象化的实体怪物。

## 二、 核心系统机制
### 1. 隐性理智值 (Sanity)
- **机制**：后台隐藏数值，不显示具体 UI 条。
- **表现**：
  - **视觉**：理智值降低时，FOV 视锥被压缩，画面出现色差、暗角或病态滤镜，视野边缘闪过幻影。
  - **听觉**：FMOD 混音改变，雨声扭曲，出现窃窃私语或错位的脚步回音。
  - **交互**：手机短信或物品描述出现乱码或恶意文本，角色移动和交互动作变得迟缓颤抖。

### 2. 房屋侵蚀度 (Environment Erosion)
- **机制**：作为“污染度”的落地表现，作用于玩家的避风港（出租屋）。
- **表现**：随着天数推进，房间出现黑色霉斑、渗出粘液、长出异物。
- **玩法**：玩家必须消耗搜集来的资源（胶带、毛巾、漂白水等）去封堵和清理侵蚀点。如果侵蚀度过高，安全屋失效。这是逼迫玩家出门探索的核心驱动力。

### 3. 无 RPG 加点系统 (基于物品的成长)
- **机制**：为了保持沉浸感和生存压力，游戏不包含传统的经验值和技能加点系统。
- **成长方式**：角色的能力提升完全依赖于探索获取的物理道具（如：更亮的强光手电、能降低侵蚀的防护服、能开锁的工具等）。

## 三、 人物设定与叙事结构
### 1. 主角（固定身份）
- **设定**：放弃“白板”或多选身份，钦定一个有具体职业和生活痕迹的普通人（如：外包程序员/独立开发者，或夜班店员等，待最终确认）。
- **心理弱点**：与理智值挂钩（如幽闭恐惧症或强迫症），在特定环境下理智值下降更快。

### 2. 女友（情感锚点与悬念核心）
- **设定**：职业可能是护士或社区工作者，比主角更早接触异变真相。
- **叙事作用**：
  - 前期作为信息源和心理安慰。
  - 中期信息逐渐诡异化（如语音背景音异常、照片无倒影）。
  - 感情戏通过碎片化环境叙事展现（如合照模糊、便利贴文字扭曲），避免大段 CG。

### 3. 双线/多视角叙事
- **机制**：在特定剧情节点，玩家视角从男主切换到女主（时间线可能倒退）。
- **体验差异**：
  - 男主视角：出租屋防守、筒子楼探索、资源管理。
  - 女主视角：外部广阔场景（如医院、街道）、纯粹的潜行、逃生与解谜。
- **悬念制造**：利用视角切换制造信息差（如玩家知道女主已遭遇不测，但切回男主视角时男主仍不知情）。

## 四、 场景设计：出租屋 (安全屋)
采用紧凑的“一室一厅一卫”格局，每个区域承担特定功能：
- **玄关与客厅**：防盗门与猫眼（核心交互与恐惧源）、电视机（单向接收外界规则）、沙发（查看手机的固定点）。
- **厨房区**：冰箱（存放食物，恢复理智/体力）、水槽（前期水源，后期侵蚀重灾区）。
- **卫生间**：镜子（心理恐怖演出，如倒影延迟）、马桶/浴缸（规则怪谈高频触发地）。
- **卧室**：床（存档/推进时间）、窗户与窗帘（必须拉上才能睡觉，隔绝外界灰雨）。

## 五、 前期剧本大纲（Day 1 - Day 3）
### Day 1：封控与规则（建立基调）
- **白天（餐厅归来）**：男主从突降灰雨的餐厅逃回出租屋。查看手机，业主群发布封控通知与规则（“拉上窗帘，绝对不要开门”）。在明亮安全的屋内吃外卖，睡觉。
- **夜晚（第一次试探）**：半夜被沉重的敲门声惊醒。触发“猫眼事件”：门外是正常的邻居大叔借盐，随后大叔脖子以诡异角度折断，但仍保持礼貌微笑。玩家理智值首次下降，建立对门外的恐惧。

### Day 2：侵蚀与修补（引入核心机制）
- **白天（房屋侵蚀）**：客厅灯光闪烁，卫生间水槽溢出黑色腥味粘液（侵蚀度上升）。女友发信息表示她那边也在渗水。玩家需在屋内寻找旧毛巾/胶带封堵水槽。大喇叭播报领取物资的新规则。
- **夜晚（高压取物）**：门外传来重物放下和机械的猫叫声（规则提示）。玩家必须打破心理防线，开门一条缝，在漆黑的走廊中拿取地上的物资袋，随后立刻关门。

### Day 3：破局与出舱（强制探索）
- **白天（资源断绝）**：停电，只能使用消耗电池的手电筒。防盗门长出霉斑，冰箱食物化为灰烬。大喇叭播报扭曲的语音：“今日无物资，请自行消化”。
- **夜晚（逼上梁山）**：收到女友最后一条带有白噪音的求救语音（“它进来了...”）后失联。隔壁传来惨叫和砸墙声，墙壁霉斑迅速扩大。玩家面临饿死或被侵蚀的绝境，被迫带上手电筒和简易武器，首次推开防盗门，踏入异变的筒子楼走廊。
